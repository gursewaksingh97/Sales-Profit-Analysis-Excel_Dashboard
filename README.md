# 📊 Sales & Profit Analysis Excel Dashboard

## Overview
This project presents an interactive Sales & Profit Analysis Dashboard built in Microsoft Excel. The dashboard helps analyze sales performance, profitability, customer trends, and business targets through data-driven visualizations.

## Dataset
The dataset contains sales transaction records along with regional performance, customer metrics, profit figures, satisfaction scores, and target achievement data. It was cleaned and transformed before analysis to ensure accurate reporting.

## Key Features
- Data Cleaning & Preparation
- Pivot Tables & Pivot Charts
- KPI Tracking
- Sales & Profit Analysis
- Customer Insights
- Interactive Dashboard

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Cleaning
- Dashboard Design
